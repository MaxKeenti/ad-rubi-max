Task 06 - Melanie - Final docs and screenshots

Owner: Melanie
Estimated effort: ~1.5 hours
Prerequisites: runnable app, 05-max-tests-and-demo-script
Day: 2-3
Blocks: submission packaging

Goal

Prepare the visible demo evidence and refresh docs so the submitted Program 2 state matches the implemented app.

Inputs

  • docs/implementation_plan.html
  • docs/guia-pruebas-programa2.html
  • Running app on emulator or device

Outputs

Create or update:

  • screenshots for normal and negative cases, if a screenshot folder exists or is created,
  • docs/guia-pruebas-programa2.html with pass/fail results,
  • docs/implementation_plan.html if implementation differs from the plan,
  • README.html with current build/run instructions.

Acceptance Criteria

  • Normal motivational chat screenshot captured.
  • Negative/crisis resource screenshot captured without graphic text.
  • Manual test runbook has results.
  • Build/run commands are accurate.
  • Docs do not claim features that are not implemented.

Pitfalls / Notes

  • Hide or crop any API key material.
  • Keep screenshots focused on the app, not IDE logs.
  • If formal Typst deliverables are later required, use Program 1 docs/entrega conventions as the template.
